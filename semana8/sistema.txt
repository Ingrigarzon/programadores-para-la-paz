El comando uname -a se utiliza en sistemas Linux para mostrar información del sistema operativo, la versión del kernel y otros datos técnicos. En Windows puede no estar disponible de forma predeterminada.
El comando whoami muestra el nombre del usuario que está utilizando el sistema en ese momento.
Conocer esta información es útil al administrar servidores porque permite identificar el sistema operativo y verificar qué usuario está ejecutando las tareas o configuraciones del sistema.
