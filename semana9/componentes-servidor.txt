Sistema operativo:
Es el software principal que administra los recursos del computador y permite ejecutar aplicaciones. También gestiona memoria, archivos y procesos.
Node.js:
Es un entorno de ejecución que permite correr código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.
Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y la forma en que responde a las solicitudes.
Puerto:
Es un canal de comunicación utilizado por el servidor para recibir solicitudes. En esta práctica se utiliza el puerto 3000.
Logs:
Son registros de eventos y actividades del sistema. Ayudan a monitorear el funcionamiento y detectar errores.
