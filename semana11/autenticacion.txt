El login permite identificar a un usuario antes de darle acceso a ciertas funciones dentro de una aplicación. Algunas rutas deben estar protegidas porque pueden contener información privada o permitir acciones importantes. Esto evita que personas no autorizadas accedan a los datos. No se deben exponer públicamente contraseñas, documentos de identidad, direcciones, teléfonos, claves de API ni otra información sensible. Así se protege la privacidad y la seguridad de los usuarios.
